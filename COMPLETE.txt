🚀 HOW TO USE

WEB INTERFACE:
  1. cd ~/Projects/inkclone
  2. source venv/bin/activate
  3. python3 web/app.py
  4. Open http://127.0.0.1:8000
  5. Type text, select options, click Generate
  6. View document preview (3-5 seconds)

COMMAND LINE:
  python3 cli.py generate "Your text" \
    --paper college_ruled \
    --ink blue \
    --artifact scan

REST API:
  curl -X POST http://127.0.0.1:8000/generate \
    -H "Content-Type: application/json" \
    -d '{
      "text": "Your text",
      "paper": "blank",
      "ink": "black"
    }'
