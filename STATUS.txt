SAMPLE COMMANDS
===============

# Generate document with college ruled paper and blue ink
python3 cli.py generate "Hello World" --paper college_ruled --ink blue

# Generate with scan simulation (realistic)
python3 cli.py generate "Important Letter" --artifact scan

# Generate with phone photo simulation
python3 cli.py generate "Quick Note" --paper legal_pad --artifact phone

# Neat handwriting
python3 cli.py generate "Formal Text" --neatness 0.9 --artifact clean

# Messy handwriting
python3 cli.py generate "Quick jot" --neatness 0.1

# Reproducible output (same seed = same randomness)
python3 cli.py generate "Test" --seed 42

# Generate template for glyph extraction
python3 cli.py template

# Extract glyphs from filled template
python3 cli.py create-profile ~/template.jpg
